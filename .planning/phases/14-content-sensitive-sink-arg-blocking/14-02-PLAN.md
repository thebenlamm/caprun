<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| executor decision → broker control plane | The plural block decision is consumed by server.rs and persisted; dropping an anchor here loses durably-recorded blocked authority. |
| broker → audit DAG (hash chain) | A type-shape change to `Event` can silently alter the hashed payload bytes of UNRELATED event types. |
| pending confirmation → human confirm UX | A plural block displayed as a single arg would let a human confirm less than they think (deferred to Phase 16, guarded fail-closed here). |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-14-05 | Tampering | `runtime-core::event.rs` serialization | high | mitigate | Preserve `skip_serializing_if = "Vec::is_empty"` so non-block events serialize byte-identically; re-run the unchanged golden-byte fixture to prove the audit hash chain is undisturbed for unrelated event types. |
| T-14-06 | Information Disclosure (unconfirmed authority) | `server.rs` block-literal side-table write | high | mitigate | Persist EVERY blocked arg's literal (iterate the anchors collection), so no blocked value is dropped before the human confirm/deny decision. Covered by durable_anchor tests. |
| T-14-07 | Elevation of Privilege (fail-open) | `audit.rs` Defect-B guard | high | mitigate | Keep the guard fail-closed with the plural shape: reject a `sink_blocked` event whose `anchors` is empty. |
| T-14-08 | Information Disclosure (partial confirm display) | `confirmation.rs::render_block_display` | medium | accept (interim) | Fail-closed assert/panic on a genuinely-plural block rather than silently showing one of N; full multi-arg narration is Phase 16 (CONFIRM-04). Accepted as interim because the single-blocked-arg case (CONTROL-02) is unaffected and no data is silently dropped. |

No package-manager installs in this plan — no supply-chain (T-*-SC) row applies.
