## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| policy gate (pre-I2) → executor I2 | policy may narrow which sinks are callable but can NEVER disable I2 (POLICY-02) |
| broker → remote transport | the destination pin (frozen redirect-none client) is the only egress; a 3xx must be refused |
| credential custody | the push token lives in broker-local env ONLY — never a ValueNode / audit literal / child / broker log |
