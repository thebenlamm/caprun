| broker → kernel | `openat2` resolves the path against the WorkspaceRoot dirfd anchor |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-33-02 | Tampering | write_within creating a new file (O_CREAT confusion with file.create) | high | mitigate | O_WRONLY\|O_TRUNC only, no O_CREAT; `write_within_missing_target_enoent` behavioral test proves fail-closed ENOENT |
