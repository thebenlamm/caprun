## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| confined worker → broker (plan-node submit) | Worker submits `PlanNode{sink, args}` for the two new sinks; untrusted values (incl. exec-output handles) cross here toward sensitive slots. |
| broker → exec child (fork/pre_exec/exec) | NEW boundary: the broker spawns arbitrary, non-caprun code as a kernel-confined child; the child's captured output re-enters as untrusted-tainted bytes. |
| exec child → kernel (Landlock/seccomp) | The child's filesystem/network/exec authority is kernel-enforced by the new narrow-allow ruleset + reused seccomp filter. |
