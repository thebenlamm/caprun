|----------|-------------|
| hostile document → confined worker (raw read) | Untrusted bytes carrying the injection + attacker address enter here; must become tainted handles, never trusted. |
| LlmPlanner → OpenAI (real call) | The injection-primed model complies; its handle choice must not be able to bypass the executor. |
| executor → SMTP adapter (email.send) | The only egress; a tainted `to` must Block before any send. |
