## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator CLI → broker | the `caprun run` operator types the intent + optional `--policy` path; the intent literal is trusted ONLY because it is operator-typed. |
| file/stream/env source → broker | any ingested content is untrusted and must be minted TAINTED, never through the trusted intent mint (M7). |
| policy source → broker | the `--policy`/`CAPRUN_POLICY` path is bound via `bind_policy` with the F1 containment refusal (POLICY-03). |
