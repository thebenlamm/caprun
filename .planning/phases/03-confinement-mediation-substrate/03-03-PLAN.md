## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-07 | Tampering | Audit DAG modified post-write | high | mitigate | SHA-256 hash-chain (parent_hash linkage); STRICT tables; no UPDATE/DELETE in app code; verify_chain detects mutation. |
| T-03-08 | DoS | Malformed/huge IPC message crashes or OOMs broker | high | mitigate | 64 KiB max-length guard before allocation; serde_json errors mapped to Error response, never panic/unwrap. |
| T-03-09 | Information Disclosure | Broker leaks internal paths/keys in error replies to worker | medium | mitigate | Error responses carry generic messages; internal detail logged via eprintln only (ASVS V7). |
| T-03-10 | Spoofing | Session id collision / predictable session | low | accept | Session ids are UUID v4 (ASVS V3); collision probability negligible for v0. |
